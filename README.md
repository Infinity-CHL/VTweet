## Build Setup:

```bash
# Download repository:
git clone https://github.com/Infinity-CHL/VTwitter.git vtwitter

# Go to the app:
cd vtwitter

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8080/
npm run serve

# Output will be at dist/ folder
npm run build
```
